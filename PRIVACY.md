# Privacy Policy

CF Gyms Statement Exporter is a local browser extension.

## Data Collection

The extension does not collect, transmit, sell, or share personal data.

It does not send Codeforces pages, cookies, account information, handles, problem statements, or generated PDFs to `icpcassiut.org` or any other server.

## Local Processing

When you export a PDF, the extension reads the currently open Codeforces problemset page in your browser, extracts the visible problem statements, and opens an internal extension print page.

To pass data from the Codeforces page to that internal print page, the extension temporarily writes the extracted statement HTML to `chrome.storage.local`.

## Local Retention

Temporary export data is removed after a successful PDF export.

If the browser or extension is interrupted before cleanup finishes, stale export data is purged automatically on extension install and browser startup.

## Chrome Web Store Limited Use Disclosure

The extension uses Codeforces page content only to provide its single user-facing purpose: exporting the complete problemset page that the user opened into a printable PDF.

The extension does not sell, transfer, share, or use page content for advertising, analytics, profiling, creditworthiness, or any unrelated purpose.

No human at ICPC Assiut reads the page content or generated PDFs. All processing happens locally in the user's browser.

## Network Access

The extension may render images that are already referenced by the Codeforces statement page, including images hosted on `codeforces.com` or `espresso.codeforces.com`.

The extension does not make analytics, tracking, telemetry, or marketing requests.

## Contact

For privacy questions, contact ICPC Assiut at `support@icpcassiut.org` or through `https://icpcassiut.org`.
