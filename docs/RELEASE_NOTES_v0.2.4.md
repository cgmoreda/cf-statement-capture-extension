# Release Notes v0.2.4

## Single Problem Export

- Added export support for individual Codeforces problem pages.
- Added individual routes for contest, Gym, group contest, and problemset problem URLs.
- Added a `Print` action to the floating in-page panel.
- Added a popup setting to show or hide the floating page button automatically.
- Kept toolbar popup export and print available even when the floating page button is hidden.

## Chrome Web Store

- This GitHub release was not submitted to the Chrome Web Store.
- Chrome Web Store version `0.2.3` remains pending review.
